Welcome to AskPilot! By Bentex

To start using AskPilot, you need to provide an API key from OpenRouter.

How to get your OpenRouter API Key:

  1. Go to the OpenRouter API Keys page.
  2. Log in or create an account.
  3. Generate a new API key.
  4. Copy the generated API key.

How to add the API Key to AskPilot:

  1. Open the AskPilot popup by clicking the extension icon.
  2. Click on the " âš™ Settings " tab.
  3. Paste your API key into the "OpenRouter API Key" field.
  4. Click the "Save Settings" button.

Once your API key is saved, you can start asking questions about the web page content!
